# Lyrics Aware Smart Music Application - Backend

Backend service for Lyrics Aware Smart Music Application. Provides APIs for song management, lyrics handling, user authentication, and music-related functionalities.

## Features

- REST API architecture
- Authentication and authorization
- Song and lyrics management
- Database integration
- Error handling middleware
- Secure API endpoints
- Modular folder structure

## Tech Stack

- Node.js
- Express.js
- MongoDB / MySQL
- JWT Authentication
- bcrypt
- dotenv
